## What is GitHub Pages?

GitHub Pages is a free service from GitHub that lets you **turn repositories into websites**.  
It takes the files in your repo (like this `index.md`) and publishes them online as a webpage.  

- You can use it for **personal portfolios**, **project documentation**, or even full websites.  
- It supports Markdown, HTML, CSS, and Jekyll themes.  
- Every time you push changes to your repo, your site updates automatically.
